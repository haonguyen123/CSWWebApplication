Text of query constraint in the predicate language identified by the CONSTRAINT‌LANGUAGE (FIQL)
The Feed Item Query Language (FIQL, pronounced “fickle”) is a simple but flexible, URI-friendly syntax for expressing filters across the entries in a syndicated feed.

Example: title == "*LC*"
Record search, Spatial search, Temporal search.
- Record search: Comma separated list of record identifiers to retrieve.
- Spatial search (bbox or geometry) : Identifies a bounding box to be used as a spatial predicate.
- Temperal Search (time) : A time instance or time period.
